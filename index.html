¿Qué son los gestores de contenido web?

El gestor de contenidos web o también conocido con sus siglas en inglés como CMS (Content Management System) es una aplicación web que permite crear, administrar y publicar una página web en internet sin la necesidad de tener conocimientos de programación o de acceder y modificar el código del sitio, tal y como utilizamos Wix para la creación de esta página, ya que brinda una interfaz de usuario intuitiva que ayuda a realizar dichas tareas a través de un panel de administración o interfaz.
